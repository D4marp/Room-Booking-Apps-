‚ùå TROUBLESHOOTING
================================================================================

PROBLEM: Schedule masih kosong / loading terus
SOLUSI:
  - Cek logcat untuk error message
  - Jika "permission-denied": Rules belum di-publish
  - Jika "failed-precondition": Index belum Enabled
  - Jika no error tapi data tidak muncul: booking mungkin empty

PROBLEM: Book Now form tidak bisa di-submit
SOLUSI:
  - Cek user sudah login
  - Cek date/time sudah di-set
  - Lihat logcat untuk error detail

PROBLEM: Booking terkirim tapi tidak muncul di schedule
SOLUSI:
  - Tunggu 2-3 detik, schedule auto-refresh
  - Atau swipe back dan buka room lagi
  - Atau hot restart app

PROBLEM: Data ada di Firestore tapi schedule masih kosong
SOLUSI:
  - Hot restart: R di terminal
  - Atau full rebuild: flutter clean && flutter run
  - Cek Firestore Rules ENABLED
  - Cek Index Status: ENABLED
